Connexion Utilisateur

GET

log/{login}/{pwd}

Connexion utilisateur avec la méthode GET.

API endpoint

                
                  http://etudiant12.chezmeme.com/rest/v1/web/mesfrais/log/{login}/{pwd} (GET)

Exemples de réponse
              
                {"ret": 0,"token": ""} //Le login a échoué
				        {"ret": 1,"token": "0d4520716134c2dce719e6f9162abfe1"} //Le login a réussi

Notes

GET

notes/{token}

Affiche la liste des notes avec la méthode GET.

API endpoint

                
                  http://etudiant12.chezmeme.com/rest/v1/web/mesfrais/notes/{token} (GET)

Exemple de réponse
              
                [
                  {
                    "note_id" : 1,
                    "note_comment" : "Formation rennes 1",
                    "note_verrou" : 0,
                    "note_usr_id" : 13,
                    "note_date" : "2018-01-12"
                  }
                  ...
                ]

                Sinon 
                  {"ret":"expired"} //Token invalide
GET

notes/{token}/{id}

Affiche le détail d'une note avec la méthode GET par son id.

API endpoint

                
                  http://etudiant12.chezmeme.com/rest/v1/web/mesfrais/notes/{token}/{id} (GET)

Exemple de réponse
              
                [
                  {
                     "note_id" : 1, 
                     "note_comment" : "Formation rennes 1", 
                     "note_verrou" : 0, 
                     "note_usr_id" : 13, 
                     "note_date" : "2018-01-12"
                  }
                ]

                Sinon
                   {"ret":false} //Si token inconnu
                   ["Pas de notes"] //Si aucune note
POST

notes/{token}

Ajouter une note avec la méthode POST.

API endpoint

                
                  http://etudiant12.chezmeme.com/rest/v1/web/mesfrais/notes/{token} (POST)

Valeurs à transmettre
              
                 "$_POST['verrou']" 
                 "$_POST['comment']" 
                 "$_POST['date']"
Exemples de réponse
              
                {"result" : 10} // Insertion effectuée
                {"ret" : "expired"} // Token inconnu
